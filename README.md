# melon-utils

A collection of minimal yet powerful **Rust** and **Python** utilities.  
I’ll keep each utility tiny, well-documented, and dependency-light.

## What’s inside (roadmap)
- **Rust**
  - `melon-log`: tiny logging helper with colorful levels and opt-in timestamps
  - `melon-args`: micro argument parser for small CLIs
- **Python**
  - `melon-color`: ANSI color helpers for quick scripts
  - `melon-timer`: tiny context manager for measuring code blocks

## Why this exists
I needed a grab-bag of tiny, reusable snippets that don’t require pulling in heavyweight frameworks.

## Contributing
Contributions are welcome—especially small, focused utilities:
1. Open an issue with your idea or bug.
2. Keep utilities self-contained and well-tested.
3. Prefer zero or minimal dependencies.
